# Smart Attendance Analytics System

A modern full-stack **Smart Attendance Management and Analytics Platform** built to manage student attendance, analyze attendance patterns, generate reports, and provide an interactive dashboard experience.

## 🚀 Live Demo

Frontend Deployment:

https://smart-attendance-qmta.vercel.app/

---

# 📌 Project Overview

Smart Attendance Analytics System is a web-based attendance management application designed for institutions and organizations.

The system helps administrators manage students, record attendance, monitor attendance percentage, visualize analytics, and generate reports through an interactive dashboard.

---

# ✨ Features

## 🔐 Authentication System

- User login
- Student and Admin roles
- Secure protected routes
- Logout functionality


## 👨‍🎓 Student Management

Admin can:

- Add students
- View student details
- Edit student information
- Delete students
- Search students


Student information includes:

- Student ID
- Name
- Email
- Phone number
- Department
- Class
- Academic details


## 📅 Attendance Management

Features:

- Mark attendance
- Present / Absent status
- Attendance history
- Date-based tracking
- Attendance percentage calculation


## 📊 Analytics Dashboard

Interactive dashboard with:

- Attendance overview
- Monthly attendance trends
- Present vs Absent statistics
- Student performance insights
- Attendance percentage cards


## 📄 Reports

Generate attendance reports:

- Individual student reports
- Overall attendance summary
- Attendance statistics


## 🎨 Modern UI/UX

The application includes:

- Responsive design
- Modern dashboard layout
- Smooth animations
- Interactive buttons
- Cards and charts
- Dark/light theme support
- Toast notifications
